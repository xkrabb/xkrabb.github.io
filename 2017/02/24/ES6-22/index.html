编程风格--笔记

发表于 2017-02-24 更新于 2024-07-01

块级作用域

  • let取代var，完全可以取代
  • 全局常量和线程安全，优先用const

字符串

静态字符串是使用单引号或反引号，动态字符串用反引号

解构赋值

  • 数组成员对变量赋值，函数参数，函数返回多个值

对象

  • 单行定义对象结尾不要逗号，多行定义使用逗号结尾
  • 对象尽量静态化，不要添加新属性，如果必须用Object.assgin
  • 尽量简洁表达

数组

  • 使用...拷贝数组，Array.form转换成数组

函数

  • 简单，单行，不回复用的函数，尽量使用箭头函数
  • 箭头函数取代Funciton.prototype.bind,不再需要额外绑定this
  • 函数体内用rest运算符...，代替agruments，使用默认参数设置

Map结构

  • 模拟实体对象用Object，而简单的key,value使用Map

Class

  • 总数用Class，取代需要prototype的操作； extends实现继承

模块

  • 用import取代require，export取代module.exports
  • 模块只有一个输出用export default，尽量不要使用*通配符
  • 输入函数小写开头，输出对象大写开头
